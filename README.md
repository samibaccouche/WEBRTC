## 🚀 How to Start

### 1. Install Dependencies

Make sure you have [Node.js](https://nodejs.org/) installed. Open your terminal in the project folder and install the WebSocket library:

```bash
npm install ws

```

2. Start the Signaling Server
   The server handles the "handshake" between the two callers.

Bash
node server.js

3- Open
index.html on your browser
